🚀 Sales Analytics Dashboard
📊 Overview

This project presents an end-to-end Sales Analytics Pipeline, transforming raw data into actionable business insights using SQL, Python, and Power BI.

The dashboard focuses on revenue, cost, and profitability trends to identify performance gaps and optimization opportunities.


🎯 Objective

To analyze sales performance and answer key business questions:

1) Is revenue growth translating into profit?

2) What are the major cost drivers?

3) Which products contribute the most to revenue?

4) Are there inefficiencies affecting profitability?


🛠️ Tech Stack

- SQL (PostgreSQL) → Data cleaning & transformation

- Python (Pandas, NumPy, Matplotlib) → EDA & analysis

- Power BI → Dashboard & visualization


📁 Dataset

AdventureWorks Sales Dataset (CSV)
Includes product, sales, and transactional data


⚙️ Workflow

1. Data Ingestion - Imported raw CSV data into PostgreSQL

2. Data Cleaning & Transformation - Removed null values and inconsistencies

- Created structured tables

- Built aggregated views for analysis

3. Exploratory Data Analysis (Python)

- Revenue trends over time

- Cost vs profit analysis

- Identification of loss-making periods

4. Dashboard Development (Power BI)

- KPI cards: Revenue, Cost, Profit, Margin

- Revenue vs Cost trend analysis

- Profit trend visualization

- Top 10 products by revenue

- 📸 Dashboard Preview



📈 Key Insights

- Revenue increased significantly, but rising costs eroded profitability

- High-revenue periods coincided with declining margins

- Indicates operational inefficiencies impacting profit


💡 Business Impact

This analysis helps:

- Identify cost inefficiencies

- Improve profit margins

- Focus on high-performing products

- Support data-driven decision making
